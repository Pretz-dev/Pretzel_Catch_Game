# プレッツェルの キャッチだいさくせん！ 🐕

愛犬プレッツェル（オーストラリアン・シェパード）が主人公の、レトロ8ビット風横スクロールキャッチゲーム。

**▶ 遊ぶ:** https://pretz-dev.github.io/Pretzel_Catch_Game/

## あそびかた

- 右から流れてくる動物（ひつじ・ねこ・いぬ・しか・りす）をキャッチしてポイントゲット！
- 逃すとHPが減る（HPは5まで）。全20レベル、だんだん難しくなる。
- **操作:** 矢印キー / WASD / 指でドラッグ
- **スペース か ワン！ボタン:** 「ワン！」を発射して遠くの動物もキャッチ
- 動きすぎるとバテる。寝ちゃったら連打で起こす。
- 母ちゃんのおやつでHP回復。父ちゃんが来るとゴロン。娘と息子が来たらボーナスタイム！
- ゴールデンレトリーバーを見ると思わずふせしちゃう。

単一HTMLファイル・依存なし。iPhoneやiPadではSafariで開いて「ホーム画面に追加」するとフルスクリーンで遊べます。
